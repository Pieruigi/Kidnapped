This Asset pack contains model of a Old Bell.

Textures are in .PSD(diffuse, specular) and .PNG file format (normal), resolution are 1024 x 1024. 

Folder Prefabs/Interactive contains the files needed to run the asset.

Folder Prefabs/noninteractive contains noninteractive objects. They can be used for environments your scene.

They are all configured correctly for use with Unity, just import into your project.

Hope you like it!
